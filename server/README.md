# meTTle server implementation

## Installation
1. Install all the dependencies first

    ```
    npm install 
    ```
    Make sure to also install Typescript globally
    ```
    npm install typescript -g
    ```
2. Compile the typescript files to javascript
    ```
    npm run compile
    //or
    tsc
    ```
3. Run the server 
    ```
    npm start
    ```
